# 2D Street Fighter Lite

A simple browser-based 2D fighting game built with **HTML + CSS + JavaScript**.

## Run locally

1. Open `index.html` directly in a browser, **or**
2. Start a local server:

```bash
python3 -m http.server 8000
```

Then open: `http://localhost:8000`

## Interval 1 (current)

This first interval includes:
- Two fighters (P1 and P2)
- Left/right movement + jump
- Basic attack hit detection
- Health bars + timer
- Win/Draw result message

## Next intervals you can add

- Interval 2: better sprites + animations
- Interval 3: special attacks (fireball, uppercut)
- Interval 4: sounds + round restart + character select
